What is the jMonkeyEngine?
--------------------------

The jMonkeyEngine3 is a 3-D game development framework for Java developers.
It is platform-independent, open-source and free.

This is an alpha release (May 2010). If you encounter any issues, please tell us!
http://www.jmonkeyengine.com/wiki/doku.php/report_bugs
We are looking forward to your feedback.


Running jMonkeyEngine Demos
---------------------------

Windows:   Double-click the "TestChooser.exe" file

Mac OS:    Double-click the "jMonkeyEngine3.jar" file

Linux:     java -jar jMonkeyEngine3.jar


Using jMonkeyEngine with an IDE
-------------------------------

We recommend using the jMonkeyPlatform, but you are free to work with the IDE of your choice.

- Create a Java SE project
- Add all .jar files from "/lib/*.jar" to the project's classpath
- You can find the jMonkeyEngine javadocs in "/jMonkeyEngine3-javadoc.zip"
- You can find the jMonkeyEngine source files in "/jMonkeyEngine3-sources.zip"
- Create a class that extends com.jme3.app.SimpleApplication.
