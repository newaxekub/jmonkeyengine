What is the jMonkeyPlatform?

About jMonkeyPlatform

The jMonkeyPlatform is a game development environment for the jMonkeyEngine 3.

Since jMonkeyPlatform is based on the NetBeans Platform you have access to all of the developer tools in the NetBeans IDE to create, compile and deploy Java applications. On top of that, the jMonkeyPlatform provides you with specialized plugins for creating jMonkeyEngine 3 game content. The jMonkeyPlatform can be downloaded and installed as a ready-to-use complete game development environment, rather than just a generic IDE to work with raw source code.

What Does the jMonkeyPlatform Deliver?

  • An integrated SDK for jMonkeyEngine 3 game development
  • Easy update to current plugin and jme3 versions (stable+svn) via update center
  • One unifying development platform that prevents scattered content design projects (like in jme2)
  • A platform for plugins: jMonkeyPlatform is a stand-alone download that can be extend with plugins
  • Independent development possibilities by using NetBeans platform plugin API
